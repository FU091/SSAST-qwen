資料集概況:
==========

1. 資料來源:
   - 本地路徑: D:\FS_SOUND
   - 包含關鍵詞: FS, TML
   - 文件格式: .wav (轉換為 .pt 格式供 SSL 使用)

2. 資料集劃分:
   - 訓練集: ssl_fs_tml_train.json (80%)
   - 驗證集: ssl_fs_tml_val.json (20%)

3. 預期功能:
   - 將 FS 和 TML 文件統一作為一類進行 SSL 預訓練
   - 隨機分配到訓練集和驗證集，避免數據洩漏
   - 保持資料的多樣性和平衡性

--------------------------------------------------

技術規格:
========

1. JSON 格式:
   {
     "wav": "/work/t113618009/spectrogram_pt_name/[filename].pt",
     "labels": "/m/09x0r"
   }

2. 處理流程:
   - 掃描 D:\FS_SOUND 目錄下的所有 .wav 文件
   - 篩選包含 'FS' 或 'TML' 關鍵詞的文件
   - 隨機打亂文件列表
   - 按 8:2 比例分割為訓練集和驗證集
   - 生成對應的 JSON 文件
